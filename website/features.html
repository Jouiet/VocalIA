Sécurité

Protection des données et isolation tenant avec chiffrement, rate limiting et conformité RGPD.

Isolation tenant (JWT)

Chaque client dispose d'un espace isolé, authentifié par JWT avec clés de signature uniques.

Chiffrement des données

TLS en transit, données sensibles chiffrées au repos via SecretVault avec clé de chiffrement dédiée.

Rate limiting

Protection contre les abus : limites configurables par IP, par tenant et par endpoint API.

CSP (Content Security Policy)

Headers de sécurité sur toutes les pages web et tous les services API pour prévenir les injections.

SRI (Subresource Integrity)

78 scripts CDN vérifiés par hash cryptographique — protection contre la compromission de dépendances.

Protection XSS

Échappement systématique de toutes les entrées utilisateur avec escapeHtml sur chaque rendu.

CORS restrictif

Origines autorisées explicitement listées pour chaque service, pas de wildcard en production.

Web Speech API ou ElevenLabs, transcription en temps réel.
