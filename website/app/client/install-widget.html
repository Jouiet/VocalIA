Installer le Widget

Ajoutez l'assistant vocal VocalIA sur votre site en 2 minutes

Installation simplifiée

Recommandé

Aucune connaissance technique requise. Choisissez votre plateforme :

WordPress / WooCommerce

0 code
  1. 1. Plugins > Ajouter > Téléverser
  2. 2. Activez le plugin
  3. 3. Réglages > VocalIA > collez votre Tenant ID
Télécharger le plugin

Shopify

Guidé
  1. 1. Online Store > Themes > Edit Code
  2. 2. Ouvrez theme.liquid
  3. 3. Collez le snippet avant </body>
Voir les instructions

PrestaShop

0 code
  1. 1. Modules > Module Manager > Upload
  2. 2. Uploadez vocalia.zip
  3. 3. Configure > collez votre Tenant ID
Télécharger le module

Wix

0 code
  1. 1. Settings > Custom Code
  2. 2. Add Code > Body End
  3. 3. Collez le snippet > Publish
Voir les instructions

Squarespace / Webflow

Guidé
  1. 1. Settings > Code Injection
  2. 2. Section Footer
  3. 3. Collez le snippet > Save
Voir les instructions

Magento 2

0 code
  1. 1. Uploadez le module dans app/code/
  2. 2. bin/magento module:enable VocalIA_VoiceAssistant
  3. 3. Stores > Config > VocalIA > Tenant ID
Télécharger le module

BigCommerce

Guidé
  1. 1. Storefront > Script Manager > Create
  2. 2. Location: Footer, Pages: All Pages
  3. 3. Collez le snippet > Save
Voir les instructions

OpenCart

0 code
  1. 1. Extensions > Installer > Upload
  2. 2. Extensions > Modules > VocalIA > Install
  3. 3. Edit > collez votre Tenant ID > Save
Télécharger le module

Joomla

0 code
  1. 1. System > Install > Upload Package
  2. 2. Activez le plugin dans Plugins
  3. 3. Cliquez VocalIA > collez votre Tenant ID
Télécharger le plugin

Drupal

0 code
  1. 1. Extend > Install new module
  2. 2. Activez VocalIA Voice Assistant
  3. 3. Configuration > VocalIA > Tenant ID
Télécharger le module

Google Tag Manager

Universel
  1. 1. Nouvelle balise > HTML personnalisé
  2. 2. Collez le snippet ci-dessous
  3. 3. Déclencheur : Toutes les pages > Publier
