Spécifications Techniques

Pour les développeurs qui veulent tout savoir

Compatibilité

  • Chrome 33+ (Desktop & Mobile)
  • Safari 14.1+ (Desktop & iOS)
  • Firefox 92+
  • Edge 79+
  • IE11: Mode texte uniquement

Performance

  • Bundle size: ~18KB gzipped
  • Lazy loading automatique
  • Shadow DOM isolé pour la sécurité
  • 0 dépendances externes
  • Hébergement Europe (VPS dédié)

  • Audio traité localement
  • Données chiffrées en transit et au repos
  • Conforme RGPD

API JavaScript

  • VocalIA.open() / .close()
  • VocalIA.on('message', callback)
  • VocalIA.setPersona(name)
