2. Définitions

  • Services : L'ensemble des solutions d'IA vocale proposées par VocalIA, incluant le Voice Widget et Voice Telephony AI
  • Voice Widget : Module d'assistant vocal intégrable sur site web, utilisant la Web Speech API
  • Voice Telephony AI : Service de réponse vocale automatisée par téléphone via réseau PSTN
  • Agent IA : Instance d'intelligence artificielle configurée pour répondre aux appels ou interactions vocales
  • Persona : Configuration comportementale et vocale d'un agent IA (ton, vocabulaire, domaine d'expertise)
  • Plateforme : L'ensemble des interfaces (site web, dashboard, API) permettant d'accéder aux Services

4. Description des services

4.1 Voice Widget

Le Voice Widget est un module JavaScript intégrable sur tout site web. Il permet aux visiteurs d'interagir vocalement avec un assistant IA. La reconnaissance et synthèse vocale utilisent les API natives des navigateurs (sans coût). Les réponses IA sont générées via un moteur Multi-AI avec 5 niveaux de redondance et facturées selon le plan souscrit.

4.2 Voice Telephony AI

Le service Voice Telephony AI permet de répondre automatiquement aux appels téléphoniques entrants. Ce service est facturé à l'usage (~0,10 €/minute) et nécessite une intégration avec un fournisseur de téléphonie PSTN.

  • API RESTful documentée
