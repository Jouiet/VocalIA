Add the VocalIA AI voice assistant to your WordPress site. No coding required.

== Description ==

VocalIA is an AI-powered voice assistant that speaks to your website visitors in 5 languages (French, English, Spanish, Arabic, Darija). It qualifies leads, remembers returning clients, and follows up automatically.

**Features:**

* Voice + text conversation in 5 languages
* Automatic lead qualification (BANT scoring)
* Cross-session memory (remembers returning visitors)
* WooCommerce integration (product search, cart recovery)
* 40 specialized AI personas
* Works on mobile and desktop

**Installation:**

1. Install and activate the plugin
2. Go to Settings > VocalIA
3. Enter your Tenant ID (from your VocalIA dashboard)
4. Add your WordPress domain to Allowed Origins in VocalIA
5. Done!

== Installation ==

1. Upload the `vocalia-voice-assistant` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu
3. Go to Settings > VocalIA to configure

== Frequently Asked Questions ==

= Do I need a VocalIA account? =

Yes. Sign up at [vocalia.ma](https://vocalia.ma/signup.html) to get your Tenant ID.

= Is it free? =

VocalIA offers a 14-day free trial. Plans start at 49 EUR/month.

= Does it work with WooCommerce? =

Yes! Select "E-commerce" as widget type for WooCommerce-specific features like product search and cart recovery.

= What languages are supported? =

French, English, Spanish, Arabic, and Moroccan Darija. The widget auto-detects the visitor's language.
