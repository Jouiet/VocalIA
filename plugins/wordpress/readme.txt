Add an AI-powered voice assistant to your WordPress site. Supports 5 languages and 40 industry personas.

== Description ==

VocalIA Voice Widget brings conversational AI to your WordPress site. Your visitors can speak naturally with an intelligent voice assistant that understands context, qualifies leads, and provides instant answers.

= Key Features =

* **40 Industry Personas** - Pre-configured AI personalities for dental, real estate, e-commerce, SaaS, healthcare, and more
* **5 Languages** - French, English, Spanish, Arabic (MSA), and Moroccan Darija
* **Zero API Costs** - Uses Web Speech API (browser-native)
* **RTL Support** - Full right-to-left support for Arabic languages
* **Lead Qualification** - Built-in BANT scoring for sales-ready conversations
* **Customizable** - Match your brand colors and widget position
* **Mobile Responsive** - Works on all devices
* **GDPR Compliant** - Voice data processed locally in browser

= How It Works =

1. Install and activate the plugin
2. Enter your VocalIA Tenant ID and API Key
3. Choose your industry persona
4. Customize colors and position
5. Done! Voice widget appears on your site

= Use Cases =

* **Customer Support** - Answer FAQs 24/7
* **Lead Generation** - Qualify visitors while they browse
* **Appointment Booking** - Voice-driven scheduling
* **Product Information** - Help customers find what they need
* **Multilingual Support** - Serve international audiences

= Requirements =

* VocalIA account (free tier available)
* WordPress 5.0+
* PHP 7.4+
* HTTPS (required for microphone access)

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu
3. Go to Settings > VocalIA Widget
4. Enter your Tenant ID from vocalia.ma/dashboard
5. Configure your preferences and save

== Frequently Asked Questions ==

= Do I need a VocalIA account? =

Yes, you need a VocalIA account to get your Tenant ID and API Key. Sign up at vocalia.ma - free tier includes 100 conversations/month.

= Is it free? =

The WordPress plugin is free. VocalIA offers a free tier with 100 conversations/month. Paid plans start at 29 MAD/month.

= What languages are supported? =

French, English, Spanish, Arabic (Modern Standard), and Moroccan Darija. More languages coming soon.

= Does it work on mobile? =

Yes, the widget is fully responsive. You can optionally hide it on mobile devices in settings.

= Is voice data stored? =

No. Voice is processed locally in the browser using Web Speech API. Only the transcribed text is sent to VocalIA servers for AI response.

= Can I exclude certain pages? =

Yes. In settings, you can specify page IDs or slugs where the widget should not appear.
